basic maths documentation¶

add_num()¶

Adds two given numbers.

This function prompts the user for two integers, adds them together, and prints the result. If the input is not a number, it will keep asking until valid input is provided.

Raises:

ValueError – If the user enters a non-integer value.

Returns:

None

div_num()¶

Divides two given numbers.

This function prompts the user for two integers, divides the first by the second, and prints the result. If the input is not a number, it will keep asking until valid input is provided.

Raises:

  • ValueError – If the user enters a non-integer value.

  • ZeroDivisionError – If the second number is zero.

Returns:

None
